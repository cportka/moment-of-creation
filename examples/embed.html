moment-of-creation — embedding

Three ways to drop the animation into a page. Open this file from a static server (e.g. npm run dev then visit /examples/embed.html).

1 · Single file in an iframe (zero build, works anywhere)

Point an <iframe> at the self-contained export — from this repo, your own host, or a CDN.

<iframe src="together.html"
        style="border:0;width:100%;height:100%"></iframe>

<!-- or straight from the repo via a CDN: -->
<iframe src="https://cdn.jsdelivr.net/gh/cportka/moment-of-creation/together.html">
</iframe>

2 · The web component (npm or a script tag)

<moment-of-creation> renders an isolated iframe internally, so it's contained, safe to repeat, and immune to your page's CSS. Size it with CSS. 

import { register } from 'moment-of-creation';
register();

<moment-of-creation
  preset="portal"
  mode="together"
  loop
  style="display:block;height:360px">
</moment-of-creation>

(This live element needs the library build — run npm run build:lib first; it loads ../dist/moment-of-creation.js below.)

3 · Programmatic embed()

import { embed } from 'moment-of-creation';
const hero = document.querySelector('#hero');
const handle = embed(hero, { preset: 'genesis', loop: true });
// later: handle.update({ mode: 'last', dials: { spForm: 3 } });
//        handle.destroy();
