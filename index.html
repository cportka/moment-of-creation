moment of creation · animation toolkit

A toolkit for two animations — a pure-CSS Burst and a binary Merger — combined into a Moment. Left: the Burst, looping. Middle: the Moment, the two combined. Right: the Merger. Swap reverses their order; Continue the chain folds the current Moment into a new random one. 29 parameters and 10 presets drive every window live. Source & engine →

Each window is the one overlay played in a mode (burst / merger / moment) — the same source the npm run export single files use. The page conducts all three: Swap plays the Moment Merger→Burst; Continue the chain freezes the current Moment as the left window, randomizes a new Moment on the right, and plays both combined in the middle. Controls + presets come from one source (dials.json / presets.json). Built with the engine in this repo.
