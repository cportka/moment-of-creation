The intro is two animations in sequence — a pure-CSS creation burst and a binary-merger splash. Here they loop on their own, and together as the original moment of creation. Each runs as a separate engine animation off one overlay; turn the knobs and every window updates live. Source & engine →

The two halves and the whole are one overlay played in three modes (creation / splash / full) — the same source the npm run export single files use. A knob marked “together” only shapes how the halves combine, so it drives the full window. Built with the engine in this repo.
